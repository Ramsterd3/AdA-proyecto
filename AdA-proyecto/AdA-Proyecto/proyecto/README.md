# Analisis de Algoritmos de Ordenamiento

## Requisitos

- Java 17
- Maven 3.8+

## Ejecucion

```bash
cd Antonio-proyecto
mvn compile exec:java
```

## Salida

- `datos_ordenados.csv` - Tabla con algoritmos, complejidad, tamano y tiempo
- `top_volumen.csv` - 15 dias con mayor volumen
- `datos_unificados.csv` - Datos financieros procesados
- `grafica_ordenamiento.png` - Grafica de barras verticales
